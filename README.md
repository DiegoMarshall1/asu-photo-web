## Web‑приложение ASU PHOTO (Flask)

### Что делает

- Принимает Excel‑файл (`.xlsx` / `.xls`) с ссылками на фото.
- Автоматически находит 2 столбца со строками, содержащими `saved-`
  - первый — фото «до»;
  - второй — фото «после».
- Извлекает дату/время из шаблона `saved-YYYYMMDD_HHMMSS`.
- Считает интервал между фото и добавляет справа столбцы:
  - `Дата_время_до`
  - `Дата_время_после`
  - `Интервал_мин`
- Показывает предпросмотр первых строк прямо в браузере.
- Позволяет скачать обработанный Excel‑файл.

### Как запустить локально

1. Перейдите в папку `web_app`:
   ```bash
   cd web_app
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python app.py
   ```
4. Откройте в браузере:
   - `http://127.0.0.1:5000`

Интерфейс:

- Кнопка выбора файла + кнопка «Загрузить и обработать».
- После обработки появляется кнопка «Скачать результат» и таблица предпросмотра.
